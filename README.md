# Overview: 
Tarot Realm is a modern Flask-based web application that provides an immersive tarot reading experience through an elegant user interface, animated effects, and interactive fortune features.

Designed with a mystical aesthetic and responsive design principles, Tarot Realm combines traditional tarot wisdom with modern web technologies to create an engaging and intuitive user experience.

__________________________________________________________________________________________________________________

# Features:
1.Tarot Card Knowledge
Explore all 22 Major Arcana tarot cards with detailed meanings and symbolic interpretations.

2.Random Tarot Reading
Draw a tarot card and receive personalized guidance, inspiration, and insight.

3.Daily Fortune
Generate daily fortune readings including:

- Love
- Career
- Health
- Lucky Color
- Lucky Number

4.Monthly Fortune
Receive monthly guidance and predictions related to:

- Career
- Relationships
- Finance
- Personal Growth

5.Zodiac Fortune
Select your zodiac sign and receive a customized horoscope-based fortune.

#Modern User Interface
- Glassmorphism design
- Smooth animations
- Interactive card effects
- Responsive layouts
- Dynamic backgrounds
- Floating star animations

# Installation:

Clone the repository:


git clone https://github.com/your-username/Tarot_Realm.git
cd Tarot_Realm
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the Flask application:

```bash
python app.py
```

Open your browser:

```text
http://127.0.0.1:5001
```
